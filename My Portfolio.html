Tools & builds

INTERNAL TOOL · WALMART

Image Sequence Validation Tool

A browser-based drag-and-drop tool for Walmart product content teams. Reads Product_Content*.xlsx item exports, displays every product's images in a gallery, lets teams reorder them by drag-and-drop, and writes the corrected order back into the same sheet — enriched with live IQS data (Abstract Product ID, Publish Status, Swatch Image). Also supports image audits, swatch-sheet matching, and cascading image order across GTINs.

INTERNAL TOOL · WALMART

Smart Validation

An optional validation layer that ingests the Allocated Images Sheet from content planning and cross-checks it against each product's live images. Every GTIN is automatically marked GOOD_TO_GO, NEEDS_REVIEW, or GTIN_NOT_FOUND — replacing a manual, line-by-line content audit.
